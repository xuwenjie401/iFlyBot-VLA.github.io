项目简介

在 index.html 输入你的项目文案。支持文字、链接、图片和视频。

  • 图片：放在 assets/images/，用 <img src="assets/images/xxx.jpg" alt="描述" />
  • 视频（本地 mp4）：放在 assets/videos/，用如下播放器

或嵌入 B 站/YouTube：把 iframe 粘在下方容器内即可。

如何使用（Git 工作流 + 部署）

# 1) 新建本地目录并初始化仓库
mkdir project-home && cd project-home
mkdir -p assets/images assets/videos

# 2) 把本文件保存为 index.html，按需把图片放到 assets/images，视频放到 assets/videos

# 3) Git 初始化并首个提交
git init
printf "node_modules\n.DS_Store\n" >> .gitignore

git add .
git commit -m "feat: initial project homepage"

# 4) 推送到 GitHub（示例）
#    先在 GitHub 上创建仓库 yourname/yourrepo，然后：
git branch -M main
git remote add origin https://github.com/yourname/yourrepo.git
git push -u origin main

# 5) 启用 GitHub Pages
#    在仓库 Settings ▸ Pages：
#    Source 选择 "Deploy from a branch"
#    Branch 选择 "main"，文件夹 "/(root)"，保存。
#    几十秒后访问 https://yourname.github.io/yourrepo

# (可选) 用命令行本地预览（需安装一个静态服务器）
# python3 -m http.server 8000
# 然后浏览器访问 http://localhost:8000

如果用 GitLab Pages/Vercel/Netlify：直接导入仓库，一键部署即可。

维护建议

  • 结构简洁：index.html + assets/ 即可。
  • 媒体优化：图片尽量 < 200KB，视频 h264/aac，必要时使用 poster 封面。
  • 相对路径：便于本地预览与线上一致。
  • 版本化内容：更新大图/视频时改文件名避免缓存问题。
